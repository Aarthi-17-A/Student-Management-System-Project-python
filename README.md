# Student Management System

A console-based Student Management System built using Python.

## Features

* Add Student
* View Students
* Search Student
* Delete Student
* Store student data using File Handling

## Technologies Used

* Python
* Lists
* Dictionaries
* Functions
* Loops
* Conditional Statements
* File Handling

## Project Description

This project allows users to manage student records through a menu-driven interface. Users can add, view, search, and delete student records. Student information is stored in a text file so that data remains available even after restarting the application.
